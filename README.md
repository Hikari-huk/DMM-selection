# DMM WEBCAMPコンテンツ
DMM WEBCAMPの学習コンテンツの研修課題です。

## フォルダの説明
#### web_site_practice
*【Webサイトを作ってみよう】の演習課題が入っています。
*2章と3章で説明したタグを使用しました。

#### JavaScript_practice
*【Webアプリケーションを作って見よう-1章(Javascriptの概要を理解しよう)】の演習課題が入っています。
*alert()を使ってJavascriptの基本を学びました。


####  Omikuji_application
*選考課題が入っているフォルダです。
*名前を入力して運勢を占うおみくじのアプリケーションです。
*名前を入力しなかった場合は「名無し」が名前の所に入ります。
